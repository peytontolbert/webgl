## What this is

This is an extracted table of ped movement/control-adjacent data used by the engine:
- locomotion clipsets (movement/strafe/transition/injured/sidestep)
- animation dictionary name (`ClipDictionaryName`)
- motion/task dataset names and pose matchers (which gate state machines)

You can map user inputs by tying your input-state machine (walk/run/sprint/crouch/strafe/aim/cover) to these clipsets and task datasets.
